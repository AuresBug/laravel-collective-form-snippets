# Laravel Collective Form For Visual Studio Code

[Visual Studio Code](https://code.visualstudio.com/) snippets to create [Laravel Collective Forms](https://laravelcollective.com/docs/6.0/html) inputs preformated for [Twitterâ€™s Bootstrap](http://getbootstrap.com/).

## Usage

When you're editing your view in Visual Studio Code, type the snippet shortcut followed by the <kbd>tab</kbd> key.

![Showcase with `boottext` snippet](showcase.gif)

## Available snippets

### General

|      Input       |      Shortcut     |
|------------------|-------------------|
| **Form**         | `bootform`        |
| **Form model**   | `bootformmodel`   |
